# 🚨 Error Master Doc (`errors/`)

> Where every Dojo mistake gets logged. The single most predictive artifact for exam readiness.

## Files

- [`review.md`](./review.md) — priority review: distilled, ordered by exam-weight × weakness. Read this first.
- [`doc-maestro.md`](./doc-maestro.md) — running log of mistakes from Dojo simulators (full detail).

## Rules

- **Log every wrong answer**, even when "I got the gist". Even right-but-uncertain ones go in the "I was guessing" section.
- **Format per entry:**
  ```
  [Main service] — [Key concept]
  Question: (1-line summary)
  Trap: what threw you off?
  Rule: the lesson in 1-2 lines.
  ```
- **Tag domain** when the Dojo gives a breakdown ("Design Secure Architectures", "Cost-Optimized", etc.) — helps spot weak domains.
- **Final review:** read this doc front to back the night before each Dojo and the night before the exam.
