## 🎯 Conceptos sugeridos a cubrir

- Tipos de records (A, AAAA, CNAME, Alias)
- **Alias vs CNAME** (Alias = solo AWS targets, gratis, funciona en raíz del dominio)
- Routing policies:
  - Simple
  - Weighted
  - Latency-based
  - Failover (active-passive)
  - Geolocation
  - Geoproximity (con Traffic Flow)
  - Multi-value answer
- Health checks (endpoint, calculated, CloudWatch)
- TTL: comportamiento y trade-off
- Hosted zones: public vs private
- Domain registration vs DNS service (puede ser solo DNS)
- 3rd party domain en Route 53

## 🔗 Comparativas relacionadas

- Alias vs CNAME
- Cada routing policy: cuándo cada una
- Failover vs Multi-value answer (no son lo mismo)
