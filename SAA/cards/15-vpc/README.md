# Block 15 — VPC & Networking

> **The biggest block in the SAA course.** If you master VPC, the exam is mostly secured. Focus on the stateful vs stateless distinction, endpoint types, and hybrid connectivity options.

## 🃏 Cards in this block

| # | Card | Concept |
|---|---|---|
| 01 | [VPC Foundations](./01-vpc-foundations.md) | CIDR, subnets, IGW, route tables, AZs, reserved IPs |
| 02 | [Security Groups vs NACLs](./02-sg-nacl.md) | Stateful vs stateless, allow vs deny, rule evaluation |
| 03 | [NAT Gateway & Bastion Host](./03-nat-bastion.md) | Private subnet outbound, NAT HA, SSH jump pattern |
| 04 | [VPC Endpoints](./04-vpc-endpoints.md) | Gateway (S3/DynamoDB free) vs Interface (PrivateLink) |
| 05 | [VPC Peering & Transit Gateway](./05-peering-tgw.md) | Non-transitive vs hub-and-spoke, scale, cost |
| 06 | [Site-to-Site VPN & Client VPN](./06-vpn.md) | IPSec tunnels, VGW+CGW, accelerated VPN |
| 07 | [Direct Connect](./07-direct-connect.md) | Dedicated vs hosted, VIFs, DX Gateway, resiliency patterns |
| 08 | [VPC Flow Logs & IPv6](./08-flow-logs-ipv6.md) | Traffic metadata, egress-only IGW for IPv6 |
