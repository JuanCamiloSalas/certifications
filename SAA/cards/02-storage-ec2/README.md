# Bloque 02 — Storage para EC2

> **EBS, EFS, Instance Store, FSx.** Es el bloque que más cae en preguntas de "qué storage elegir según workload".

## 🎯 Conceptos sugeridos a cubrir

- EBS volume types (gp2 vs gp3, io1 vs io2, st1, sc1)
- EBS snapshots: incrementales, copia entre regiones, FSR (Fast Snapshot Restore)
- EBS Multi-Attach (solo io1/io2)
- EBS encryption en flight y at rest
- EFS: standard vs IA, performance modes, throughput modes
- Instance Store: cuándo y por qué (ephemeral)
- FSx for Windows, Lustre, NetApp ONTAP, OpenZFS
- AMI lifecycle desde EBS

## 🔗 Comparativas relacionadas

- EBS vs EFS vs Instance Store
- EFS vs FSx (variantes)
- gp2 vs gp3 (cuándo migrar)
- FSx Lustre vs FSx for Windows vs FSx NetApp
