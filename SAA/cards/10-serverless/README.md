## 🎯 Conceptos sugeridos a cubrir

- Lambda: límites (15 min timeout, 10 GB mem, 6 MB sync payload, 256 KB async, 512 MB-10 GB ephemeral storage `/tmp`)
- Lambda triggers: síncronos vs asíncronos vs poll-based (event source mappings)
- Lambda concurrency: reserved vs provisioned, cold starts
- Lambda destinations (success/failure)
- Lambda layers
- Lambda@Edge vs CloudFront Functions
- Lambda + VPC (ENI, escalado)
- API Gateway: REST vs HTTP vs WebSocket
- API Gateway endpoints: Edge-optimized vs Regional vs Private
- API Gateway authorization: IAM, Cognito, Lambda authorizer
- API Gateway throttling y caching
- API Gateway integrations (Lambda, HTTP, AWS service, Mock)
- Step Functions: Standard vs Express
- Step Functions: estados, error handling, parallel, map
- AppSync: GraphQL managed
- Cognito User Pools vs Identity Pools

## 🔗 Comparativas relacionadas

- Lambda vs ECS Fargate vs EC2
- API Gateway REST vs HTTP vs WebSocket
- Step Functions Standard vs Express
- Lambda@Edge vs CloudFront Functions
- Cognito User Pool vs Identity Pool
