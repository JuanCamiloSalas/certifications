# 💡 Exam Tips (`tips/`)

> Trigger phrases, traps, and patterns harvested **from real Dojo questions**. This folder is the highest-ROI artifact for the final-week review.

## Files

- [`tips-examen.md`](./tips-examen.md) — running list of trigger phrases and traps, grouped by service family.

## Rules

- **Add tips AFTER each Dojo**, never in cold theory. Tips only matter if they came from a question you struggled with.
- **Format:** `> "trigger phrase" → service / answer`. One line each. No paragraphs.
- **Group by service family** (Storage, Compute, Networking…) so you can review by chunk.
- **Final-week review:** read `tips-examen.md` front to back the day before each Dojo / the exam.
