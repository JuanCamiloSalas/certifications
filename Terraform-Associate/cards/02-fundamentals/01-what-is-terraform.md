# What is Terraform?

> **Pitch (1 line):** HashiCorp's IaC tool that codifies, standardizes, and provisions infrastructure across any platform using HCL.

## 🎯 What the exam tests

- The three value props: **codify** infrastructure, **standardize** the workflow, **platform agnostic**.
- That Terraform uses **HCL** as its configuration language.
- That it's declarative and reconciles a desired state against real infrastructure.

## 🧠 Core (non-obvious bits)

- **Codify your infrastructure** — repeatable, versioned workflow instead of console clicking.
- **Standardize workflows** — one tool/one workflow for all the resources an app needs; scale via modules.
- **Platform agnostic** — same HCL provisions any cloud or datacenter through providers (not tied to one vendor).
- Terraform is the *open-source CLI/binary*; HCP Terraform is the SaaS platform built on top (see editions card).
