- ⬜ `source` types: **registry** (public/private), **Git**, **local path**. `version` applies **only** to the registry.
- ⬜ Passing **input variables** to a module and reading its **outputs**.

- ⬜ `count` / `for_each` on a module block to instantiate multiple times.
- ⬜ Public **Terraform Registry** — how to find and pin a module version.
