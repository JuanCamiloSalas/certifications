# 🖼️ Assets (`assets/`)

> Diagrams, screenshots, and any image referenced by a card or comparison.

> **Heads-up for this exam:** Terraform needs **far fewer images than SAA**. Most diagrams here (workflow, state locking, dependency graph) are conceptual → prefer **Mermaid inline** in the card. Only download a real image when a picture genuinely beats text. Expect this folder to stay light.

## Rules

- **Add an image only when it actually helps.** Decorative pictures are noise.
- **File name = self-describing.** `core-workflow.png`, not `image1.png`.
- **Reference from a card with a relative path:**
  ```markdown
  ![alt text](../../assets/<file>.png)
  ```
- **For original drawings:** mobile photos of hand-drawn diagrams are fine — crop and rename.

## Suggested subdivisions (optional)

If it grows, split by block (`assets/03-core-workflow/`, …). Don't pre-create subfolders; make them only when you have ≥ 3 images for a block.
